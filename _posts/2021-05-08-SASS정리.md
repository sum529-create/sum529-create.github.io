# Sass와 SCSS 차이점
- {}(중괄호)와 ;(세미콜론)의 유무로 다르다
- Mixins('믹스인'은 재사용 가능한 기능을 만드는 방식)

## Sass
{% highlight html %}
.list
  width: 100px
  float: left
  li
    color: red
    background: url("./image.jpg")
    &:last-child
      margin-right: -10px
{% endhighlight %}

## SCSS:
{% highlight html %}
.list {
  width: 100px;
  float: left;
  li {
    color: red;
    background: url("./image.jpg");
    &:last-child {
      margin-right: -10px;
    }
  }
}
{% endhighlight %}

# 컴파일 방법
- Sass(SCSS)는 웹에서 직접 동작 불가능
- CSS로 변환후 실행해야함

## SassMeister 사이트
<https://www.sassmeister.com/>
{% highlight html %}
.container{
    width:100px;
    /* 중첩가능 */
    .item{
        width: 200px;
    }
}
