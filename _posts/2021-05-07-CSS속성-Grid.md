# Grid
- 2차원(행과 열)의 레이아웃 시스템

# Grid Containers 속성 - display
> grid          - Block 특성의 Grid Container를 정의
> inline-grid   - Inline 특성의 Grid Container를 정의


{% highlight html %}
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
</div>
{% endhighlight %}

{% highlight html %}
.container{
  display: inline-grid; /* grid */
  width: 300px;
  height: 250px;
  border: 4px solid lightgray;
}
.item{
  border: 2px dashed red;
}
{% endhighlight %}


# grid-template-rows
- 명시적 열(Track)의 크기를 정의
- 동시에 라인(Line)의 이름도 정의
- fr(fraction, 공간 비율) 단위를 사용
ex) grid-template-columns: 1fr 1fr; /* 1 : 1비율 */
- repeat() 함수 사용 가능 -> grid-template-columns:repeat(2, 1fr);


{% highlight html %}
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
</div>
{% endhighlight %}

{% highlight html %}
.container{
  display:grid;
  grid-template-rows: repeat(2, 100px); /* 2행 */
  grid-template-columns: repeat(3, 1fr); /* 200px 200px -> 2열, 1fr 1fr -> 1:1비율 */
  border: 4px solid lightgray;
}
.item{
  border: 2px dashed red;
}
{% endhighlight %}




# Grid Items 속성 - grid-row
- grid-row-start과 grid-row-end의 단축 속성
- 각 속성을 /로 구분

# Grid Items 속성 - grid-column
- grid-column-start과 grid-column-end의 단축 속성
- 각 속성을 /로 구분

{% highlight html %}
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
</div>
{% endhighlight %}

{% highlight html %}
.container{
  width:600px;
  display:grid;
  grid-template-rows: repeat(4, 100px);
  grid-template-columns: repeat(3, 1fr);
  border: 4px solid lightgray;
}
.item{
  border: 10px solid red;
  font-size: 30px;
}
/* chrom에서 f12 해당영역 클릭후 gird클릭 */
.item:nth-child(1){
  grid-row: 1/3;
  grid-column: 1 / 3;
}
.item:nth-child(2){
  grid-row: 1/ 3;
}
.item:nth-child(4){
  grid-row: 4/5;
  grid-column: -1/-3;
}
{% endhighlight %}



# Grid Containers 속성 - grid-template-areas
- 지정된 그리드 영역 이름(grid-area)을 참조해 그리드 템플릿을 생성합니다.

{% highlight html %}
.container {
  display: grid;
  grid-template-rows: repeat(3, 100px);
  grid-template-columns: repeat(3, 1fr);
  grid-template-areas:
    "header header header"
    "main main aside"
    "footer footer footer";
}
/* grid-area로 이름을 지정하고 grid-template-areas로 지정해 준다. */
header { grid-area: header; }
main   { grid-area: main;   }
aside  { grid-area: aside;  }
footer { grid-area: footer; }
{% endhighlight %}


{% highlight html %}
<div class="container">
  <div class="item">HEADER</div>
  <div class="item">MAIN</div>
  <div class="item">ASIDE</div>
  <div class="item">FOOTER</div>
</div>
{% endhighlight %}

{% highlight html %}
.container{
  width: 400px;
  display:grid;
  grid-template-rows: repeat(3, 100px);
  grid-template-columns: repeat(3, 1fr);
  grid-template-areas:
    "header header header"
    "main . aside"
    ". footer footer"; /* 중간에 칸을 비우고 싶다면 '.'을 입력한다 */
}
.item{
  border: 10px solid red;
}
.item:nth-child(1){
  grid-area: header;
}
.item:nth-child(2){
  grid-area: main;
}
.item:nth-child(3){
  grid-area: aside;
}
.item:nth-child(4){
  grid-area: footer;
}
{% endhighlight %}



# Grid Containers 속성 - row-gap(gird-row-gap)
- 각 행과 행 사이의 간격(Gutter)을 지정합니다.

{% highlight html %}
.container {
  row-gap: 크기;
}
{% endhighlight %}

# Grid Containers 속성 - columm-gap(grid-column-gap)
- 각 열과 열 사이의 간격(Gutter)을 지정합니다.

{% highlight html %}
.container {
  column-gap: 크기;
}
{% endhighlight %}

**=> column-gap + row-gap을 한번에 사용시 gap을 사용한다.**

# Grid Containers 속성 - gap(grid-gap)
- 각 행과 행, 열과 열 사이의 간격(Gutter)을 지정합니다.

**grid는 더 이상 사용하지 않아 gap(row-gap, column-gap)으로 교체 가능**



# Grid Containers 속성 - grid-auto-columns
- 암시적 열의 크기를 정의
- 음수 사용 
- 아이템이 grid-template-columns로 정의한 명시적 열 외부에 배치되는 경우 암시적 열의 크기가 젹용된다.
- gird-template-rows와 grid-template-columns로 정의한 영역 이외에 영역의 크기를 지정하여 사용한다.

{% highlight html %}
<div class="container">
  <div class="item">1</div>
  <div class="item">2</div>
  <div class="item">3</div>
  <div class="item">4</div>
  <div class="item">5</div>
  <div class="item">6</div>
</div>
{% endhighlight %}

{% highlight html %}
.container{
  width: 600px;
  display:grid;
  grid-template-rows: repeat(2, 100px);
  grid-template-columns: repeat(3, 1fr);
  grid-auto-rows: 100px;
  grid-auto-columns: 1fr; /* 1:1:1 비율로 설정*/
}
.item{
  border:10px solid red;
}
.item:nth-child(1){
  grid-column: 1/3;
}
.item:nth-child(2){
  grid-column: 3/5;
}
.item:nth-child(3){
  grid-column: span 2;  /* 2개의 선만큼 늘어나라 -> 2개의 칸을 차지한다. */
}
/* 먼곳의 암시적 영역에서도 지정할 수 있다. 위에서 암시적 크기를 지정해놨기 때문 */
.item:nth-child(3){
  grid-row: 5/7;
  grid-column: 6/7;
}
{% endhighlight %}



# Grid Containers 속성 - grid-auto-flow
- 배치하지 않은 아이템(Item)을 어떤 방식의 ‘자동 배치 알고리즘’으로 처리할지 정의

> row               - 각 행 축을 따라 차례로 배치 (기본값)      왼 -> 오
> column            - 각 열 축을 따라 차례로 배치               위 -> 아래
> row dense(dense)  - 각 행 축을 따라 차례로 배치, 빈 영역 메움!
> column dense      - 각 열 축을 따라 차례로 배치, 빈 영역 메움!	

**row dense(dense) 예제**
{% highlight html %}
/* For row & row dense */
.container {
  display: grid;
  grid-template-rows: repeat(3, 1fr);
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: row || row dense || dense;
}
.item:nth-child(2) {
  grid-column: span 3;
}
{% endhighlight %}

*grid-auto-flow:row;*
**1 _ _
    2
  3 4 _**

*grid-auto-flow:rowdense;*
**1 3 4
    2 
  _ _ _**

**column dense 예제**
{% highlight html %}
/* For column & column dense */
.container {
  display: grid;
  grid-template-rows: repeat(3, 1fr);
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: column || column dense;
}
.item:nth-child(1) {
  grid-column: 2 / span 2;
}
.item:nth-child(2) {
  grid-column: span 2;
}
{% endhighlight %}

*grid-auto-flow: column;*
**_ _ 1
  2   _
  3 4 _**

*grid-auto-flow: column-dense;*
**3  1
  2   _
  4 _ _**


{% highlight html %}
<div class="container">
  <item class="item">1</item>
  <item class="item">2</item>
  <item class="item">3</item>
  <item class="item">4</item>
</div>
{% endhighlight %}

{% highlight html %}
.container{
  width: 600px;
  display: grid;
  grid-template-rows: repeat(3, 100px);
  grid-template-columns: repeat(3, 1fr);
  grid-auto-flow: row dense; /* row 생략가능 다만 column은 생략 불가능 row로 인식하기 때문 */
}
.item{
  border: 10px solid red;
}
.item:nth-child(1){
  grid-column: span 2;
}
.item:nth-child(2){
  grid-column: -1 / -3;
}
{% endhighlight %}
