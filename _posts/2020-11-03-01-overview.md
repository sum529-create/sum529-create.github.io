# 01_블록(Block), 인라인(Inline)


{% highlight html %}
<!-- 
	1. 블록 요소
	    1) DIV, H!, P
	    2) 사용 가능한 최대 가로 너비를 사용한다.
	    3) 크기를 지정할 수 있다.
	    4) width: 100%; height:0; 으로 시작
	2. 인라인 요소
	    1) SPAN, IMG
	    2) 필요한 만큼의 너비를 사용한다.
	    3) 크기를 지정할 수 없다.
 -->

<!DOCTYPE html> <!-- HTML5 임을 명시 -->
<html>
	<head>

		<meta charset="UTF-8">
		<title>Insert title here</title>
	</head>
	<body>
		웹 페이지 본문 
	</body>
</html>

{% endhighlight %}
